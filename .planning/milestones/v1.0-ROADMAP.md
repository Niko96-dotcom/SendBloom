# Roadmap: SendBloom

## Overview

SendBloom ships as ten vertical milestone phases, each producing a loadable, testable AU/VST3 plugin artifact. **The routing is the product** — Phase 3 delivers a deliberately crude but complete signature chain so bloom-then-chop behavior is audible before any DSP polish. Later phases swap placeholders for real IO, gates, reverb, overdrive, and pressure send while preserving proven topology. Within every phase, work follows the architecture's 20-step engineering build order as horizontal acceptance gates. No phase completes until pluginval passes at the appropriate strictness, unit tests for that phase's scope pass, and an audible DAW smoke test is documented.
