# Plan 26-01: Capture Protocol and Metrics Tooling

## Objective

Create the complete Outcome-B capture package: deterministic source generation, an auditable WAV analyzer, metadata-bearing JSON/CSV results, the full hardware grid protocol, and synthetic tests.

## Tasks

1. Write the clean-room capture protocol with equipment/calibration, deterministic stimuli, naming/metadata schema, wet isolation, full grids, rejection rules, and explicit `human_needed` handling.
2. Implement deterministic standard-library stimulus generation with fixed seed and manifest.
3. Implement standard-library PCM WAV analysis for predelay, EDT/RT estimates, spectral centroid over time, harmonics, DC, and gate envelope; emit stable JSON and CSV carrying metadata and knob settings.
4. Add unit/CLI tests using known synthetic captures and confirm prohibited inputs are neither required nor produced.

## Verification

- `python3 -m unittest discover -s tests/reference -v`
- Generate sources twice and compare SHA-256 manifests.
- Analyze a generated/synthetic WAV and validate JSON + CSV schema.

## Done When
